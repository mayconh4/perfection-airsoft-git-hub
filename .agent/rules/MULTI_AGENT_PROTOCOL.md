# 🛰️ Protocolo de Sistema Autônomo de Múltiplos Agentes (SAMA)

Este documento define o modo de operação autônomo para o projeto Perfection Airsoft.

## 🛠️ Funcionamento do Sistema
1. **Análise de Contexto**: Toda solicitação é processada para identificar a especialidade requerida.
2. **Roteamento Inteligente**:
   - **Frontend/UI/UX**: `frontend-specialist`
   - **Backend/Scripts/APIs**: `backend-specialist` / `devops-engineer`
   - **Automação/Processos**: `orchestrator` / `automation`
   - **Marketing/SEO/Copy**: `seo-specialist` / `copywriter`
   - **Segurança**: `security-auditor` / `penetration-tester`
   - **Planejamento**: `project-planner` / `product-manager`
3. **Execução Direta**: Tarefas são executadas imediatamente sem pedidos de confirmação ou autorização.
4. **Prioridade**: Velocidade, Eficiência e Resultado Prático (Foco em ROI e Automação).

## ⚖️ Regras de Decisão
- Especialização Máxima.
- Abordagem Técnica e Direta.
- Zero Interrupção de Fluxo.
- Sem explicações desnecessárias.
